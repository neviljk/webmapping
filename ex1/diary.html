Sikkim, an enchanted land
Development Diary

APRIL 4, 2016

Searching for tools

Finalized the topic. Looked at recommendations for IDEs and settled on Brackets by Adobe.

APRIL 7, 2016

Looking for design ideas

Checked out several travel blogs in order to get some ideas for the design. Most of them used Javascript so I had to think of a basic interface. I liked the Playfair font used in theartofsimple.net

APRIL 9, 2016

Layout

I decided to go for a layout with the navigation menu on the left, and a simple header on top. I created a simple header using my own photograph. As for the fonts, I narrowed down to a Serif based and integrated it using Google WebFonts.

APRIL 11, 2016

Positioning Problems

The website title was not getting positioned at the center of the header. I learned line-height argument could be used to overcome this problem. I could not figure out a way to get the current page to be highlighted.

APRIL 13, 2016

CSS

I worked on configuring the CSS for styling the menu text, heading and sub-heading and bolded elements.

APRIL 14, 2016

Content

Added content for the website and collected photographs from personal and external sources. Had to fine tune CSS as I added content. Created CSS classes for position the images on the left and right. Faced diffulty in getting the captions to align properly underneath the images.

APRIL 17, 2016

Fine tuning

Invalid characters were in the footer. Used HTML entities to solve the problem.
